# 📝 AI Text Summarizer

A sleek, modern web application that leverages Puter.js and AI to instantly summarize long passages of text. Built with React and styled with Tailwind CSS, featuring a deep, "space-grade" dark aesthetic.

## ✨ Features

* **Instant Summarization**: Powered by the Puter.js AI Chat API.
* **Real-time Initialization**: Automatically detects and connects to the Puter environment.
* **Modern UI**: High-end glassmorphism design with Tailwind CSS gradients.
* **Responsive**: Optimized for both mobile and desktop screens.

## 🚀 Tech Stack

* **Framework**: [React.js](https://reactjs.org/)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/)
* **Cloud Ecosystem**: [Puter.js](https://puter.com/)
* **Bundler**: [Vite](https://vitejs.dev/)


## 📖 How it Works

1.  **Initialization**: The app polls for the `window.puter.ai` object. Once found, the status indicator turns green ("AI is ready").
2.  **Input**: Paste your long-form text into the glass-morphic text area.
3.  **Process**: Click **Summarize**. The app sends a structured prompt to the Puter AI model.
4.  **Result**: The AI returns a concise summary, displayed in a clean, readable card.
